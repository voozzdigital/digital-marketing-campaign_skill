# Links Patrocinados — Guia de Copy para Campanhas

## Plataformas cobertas
- **Google Ads** (Search, Display, Performance Max)
- **Meta Ads** (Facebook + Instagram — Feed, Stories, Reels)

---

## Google Ads — Rede de Pesquisa

### Estrutura de um anúncio responsivo (RSA)

```
HEADLINES (até 15 opções — Google testa automaticamente)
  └─ Máx. 30 caracteres cada
  └─ Inclua: keyword principal, benefício, diferencial, urgência
  └─ Google escolhe as melhores combinações

DESCRIPTIONS (até 4 opções)
  └─ Máx. 90 caracteres cada
  └─ Expanda o benefício, adicione prova social ou garantia
  └─ Termine com CTA

URL DISPLAY
  └─ Mostre a URL de forma limpa: exemplo.com/consultoria
```

### Headlines que convertem (exemplos por objetivo)

**Tráfego / Awareness:**
- "Aprenda [tema] do Zero — Grátis"
- "Guia Completo de [tema] — Baixe Agora"

**Geração de Lead:**
- "Consultoria Gratuita de [serviço]"
- "[X] Vagas Disponíveis — Reserve a Sua"
- "Diagnóstico Grátis para [segmento]"

**Venda Direta:**
- "[Produto] com [X]% de Desconto"
- "Entrega em 24h — [Produto] Disponível"
- "Frete Grátis + Garantia de [X] dias"

### Descriptions que convertem
- "Ajudamos [número] empresas a [resultado]. Solicite orçamento sem compromisso."
- "Sem contrato. Sem fidelidade. Resultados em [prazo]. Fale com especialista."
- "[Depoimento curto de cliente]. Veja como funciona — é grátis."

### Extensões essenciais (não pule)
- **Sitelinks**: 4 links adicionais (ex: "Como Funciona", "Preços", "Depoimentos", "Contato")
- **Callouts**: diferenciais em 25 chars ("Suporte 24h", "100% Online", "Sem Taxa de Adesão")
- **Snippets**: lista de serviços/produtos
- **Formulário de lead**: captura direto no Google (sem landing page)

## Meta Ads (Facebook + Instagram)

### Estrutura de um anúncio

```
HEADLINE (título sob o criativo)
  └─ Máx. 40 caracteres recomendados (pode ser mais)
  └─ Foco: o resultado ou a oferta

PRIMARY TEXT (texto acima do criativo)
  └─ Máx. 125 chars para não truncar no feed
  └─ Hook nas primeiras 2 linhas
  └─ Desenvolva em 3–5 linhas
  └─ CTA no final

DESCRIPTION (linha abaixo do headline)
  └─ Complemento ou urgência
  └─ Reforço do CTA

CTA BUTTON
  └─ Escolha: Saiba Mais | Comprar | Cadastrar | Baixar | Enviar Mensagem
```

### Fórmulas de Primary Text que convertem

**Fórmula PAS:**
```
[Problema em 1 frase]
[Agravante — o que acontece se não resolver]
[Solução — seu produto/serviço]
[CTA]
```

**Fórmula de Prova Social:**
```
"[Nome/cargo] estava [problema]. Depois de [tempo] com [produto/serviço], [resultado concreto]."
[CTA]
```

**Fórmula de Oferta:**
```
"Só até [data/quantidade]: [oferta específica]."
[Benefícios em bullets: ✅ X ✅ Y ✅ Z]
[CTA]
```

## Criativos: o que funciona em 2025

### Para Meta Ads
| Formato | Desempenho | Melhor para |
|---|---|---|
| Vídeo 9:16 (Reels/Stories) | ⭐⭐⭐⭐⭐ | Awareness, produto físico |
| Carrossel | ⭐⭐⭐⭐ | E-commerce, features |
| Imagem estática | ⭐⭐⭐ | Oferta direta, lead |
| UGC (vídeo de usuário) | ⭐⭐⭐⭐⭐ | Conversão, BOFU |

### Regras de criativo
- Texto na imagem: máx. 20% da área (Meta penaliza mais)
- Primeiros 3 segundos do vídeo: decisivos — comece com movimento ou fala
- Legenda nos vídeos: sempre (85% assiste sem som)
- Rostos humanos: aumentam CTR significativamente

## Métricas para acompanhar

### Google Ads
- **CTR**: > 5% é bom para search
- **CPC**: compare com benchmark do setor
- **Taxa de conversão**: > 3% é razoável
- **Quality Score**: > 7 reduz CPC

### Meta Ads
- **CTR (link)**: > 1% é bom
- **CPM**: monitore para ver saturação de audiência
- **ROAS**: meta mínima 3x para e-commerce
- **Frequência**: acima de 3–4 = criativo cansado, troque

## Estrutura de campanha recomendada (Meta)

```
CAMPANHA — Objetivo (Conversões / Tráfego / Leads)
  └─ CONJUNTO DE ANÚNCIOS A — Público Amplo (Advantage+)
      └─ Anúncio 1: criativo vídeo
      └─ Anúncio 2: criativo imagem
  └─ CONJUNTO DE ANÚNCIOS B — Retargeting (visitantes do site)
      └─ Anúncio 1: oferta ou urgência
      └─ Anúncio 2: prova social
```

---

## Erros mais comuns em links patrocinados
- Mandar tráfego para a home (sempre use landing page específica)
- Não instalar pixel / tag de conversão antes de ativar
- Pausar campanha cedo demais (< 7 dias) — o algoritmo precisa aprender
- Testar muitas variáveis ao mesmo tempo
- Ignorar mobile (> 70% do tráfego é mobile)
- Copy igual para todos os públicos — segmente a mensagem
